# E-Wallet Project Setup Guide
## Clone Repository
```
git clone https://github.com/zaxe17/e-wallet.git
```
## Create a New Branch
```
git branch <branch-name>
```
## Switch to Your Branch
```
git checkout <branch-name>
```
## Install Composer
```
composer install
```
## Run the Laravel Server
```
php artisan serve
```
## Run Tailwind CSS
```
npm run dev
```
